# 中段剧情历程梳理与补完建议

## 目的
这份文档只处理一个问题：

当前版本除开场序章和第 7 天结局外，中段剧情为什么玩起来不清楚，以及要补哪些内容，才能让 Act 1 在故事逻辑上完整。

这里讨论的是“故事层面的游戏历程”，不是系统重构方案。重点是：

1. 玩家当前实际会经历什么
2. 王麻子线、疯长老线为什么不清楚
3. 随机事件为什么现在很难服务主线
4. 最少需要补哪些节点和文本，才能让整段故事成立

## 一、当前 Act 1 的实际故事骨架

当前线上主骨架仍然成立，结构是：

- `1001` 第一天正式落地，明确“你已经不是会白死一次的容器，今天必须先拿到保命线索”
- `1002` 第二天早晨，王麻子公开压迫，告诉玩家外门秩序本身就是危险来源
- `1003` 第四天夜里，玩家必须第一次明确“自己要往哪条活路上押注”
- `1004` 第五天夜里，疯长老亲自下场，把“活命”和“试药”绑定
- `1005` 第六天夜里，要求玩家把前面拿到的零散线索收束成路线
- `1006` 第七天早晨，月考与结局判定

从骨架上看，这一章真正想讲的是：

- 开局先活下来
- 认清外门秩序不是背景，而是压迫机制
- 在柳飞霞、王麻子、疯长老、夜巡与化骨池异常之间找可利用的缝
- 在月考前完成选边，并带着这条路进入结局

所以大方向没有错，问题出在“中段桥接”和“开线提示”明显不足。

## 二、当前玩家实际会怎么玩到

结合 `events.csv`、`event_triggers.csv`、`story_event_scheduler.gd` 和行动系统，当前中段流程实际更接近下面这样：

1. 玩完开场后进入第 1 天白天
2. 第 1 天最优先触发柳飞霞首轮 `2001`
3. 点“入侵”进入柳飞霞首战 `9101`
4. 战后进入 `9102 / 9103`
5. 进入第 2 天早晨固定主线 `1002`
6. 第 2 天白天开始，系统每次只会取一个当前可触发的条件事件
7. 如果没有条件事件，才会给玩家 3 选 1 行动，再由行动把当前位置切到某个地点
8. 夜间和 closing 自动推进

关键问题在这里：

- 玩家不再手动移动地点
- 但底层事件调度仍然使用 `current_location_id`
- 这个 `current_location_id` 由白天三选一行动里的 `target_location_id` 暗中决定

也就是说，现在线路开启仍然部分依赖“你白天选的是哪类行动”，但界面上已经不再把这件事明确当成“去广场 / 去丹房 / 去茅草屋”来解释。

这就是目前玩家会觉得“我不知道王麻子线和疯长老线怎么开”的根本原因之一：

不是线不存在，而是开线条件仍然部分藏在底层地点语义里，玩家表层看不到。

## 三、当前中段剧情的主要问题

### 1. 柳飞霞线是清楚的，但压住了其他线

柳飞霞线当前最完整：

- `2001 -> 9101 -> 9102/9103`
- `2002`
- `2102`

它的问题不是没有内容，而是优先级过高、桥接最完整，因此会天然成为玩家最先感知到、也最容易持续跟进的一条线。

当前 `conditional_story` 只会在同一时机取一个最高优先级事件。

而现在线上优先级是：

- `2001 = 360`
- `9102/9103 = 354`
- `2002 = 350`
- `2102 = 340`
- `2004 = 330`
- `2005 = 320`
- `2003 = 310`

这意味着在柳飞霞线仍然可触发时，王麻子线、夜巡线、疯长老线会不断被压后。

从系统角度这没错，但从故事体验上会导致两个问题：

- 玩家会误以为“这一章主线只有柳飞霞”
- 玩家不容易意识到其他线也是正经路线，而不是随机插曲

### 2. 王麻子线和疯长老线缺“开线理由”

从数据上看：

- 王麻子线入口是 `2004`
- 疯长老线入口是 `2003`
- 夜巡线入口是 `2005`

它们现在的触发条件极轻，基本只有：

- 当前是白天
- 当前地点匹配

但玩家体验并不会因此更顺，反而更乱。原因是：

- 它们缺少“为什么此刻应该去碰这条线”的桥接节点
- 缺少明确告诉玩家“这条线能解决什么”的正文承接
- 入口事件正文本身还是空的或几乎空的

当前明显空缺的正文包括：

- `evt.2003.desc`
- `evt.2004.desc`
- `evt.2005.desc`
- `evt.2007.desc`
- `evt.2008.desc`
- `evt.2102.desc`

这些不是细节缺失，而是直接导致“线存在，但玩家感知不到它是线”。

### 3. 中段缺桥接节点，导致主线像跳拍

当前主骨架从 `1002` 直接跳到 `1003`，再跳到 `1004`、`1005`。

骨架在，但骨架之间缺几类很关键的桥：

- 从“王麻子开始施压”到“我为什么要主动调查他和药房”
- 从“柳飞霞首战后建立接触”到“她为什么开始递暗线”
- 从“疯长老的试药危险”到“我为什么还要继续接近他”
- 从“夜间异常越来越重”到“它和外门筛选到底有什么关系”

当前这些桥，大量被空节点、随机事件或玩家自己脑补承担了。

结果就是：

- 系统知道自己在推进什么
- 设计者知道这些线彼此相关
- 玩家感受到的却是“零散事件轮流弹出来”

### 4. 随机事件大多只有氛围，没有路线功能

当前随机事件分成几池：

- `act1_resource_pool`
- `act1_relation_pool`
- `act1_anomaly_pool`
- `act1_risk_pool`
- `act1_battle_pool`

问题不在“有随机事件”，而在“随机事件还没有承担明确叙事职责”。

现在它们大多是：

- 单段氛围
- 单项资源波动
- 单次风险提醒

能增强密度，但不能回答玩家最关心的 3 个问题：

1. 我下一步该盯谁
2. 我这一轮是在往哪条活路上走
3. 当前异常究竟是在推动哪条冲突升级

因此当前随机事件会让人觉得“有东西在发生”，但不会让人觉得“剧情更清晰了”。

### 5. 当前位置仍然影响触发，但玩家几乎无感知

这是现在最容易被忽视、但最伤叙事清晰度的一点。

虽然手动移动已经被砍掉，但白天行动仍会设置地点：

- 宿舍类行动把位置落到 `01`
- 广场类行动把位置落到 `02`
- 丹房外间类行动把位置落到 `03`
- 进一步调查才会碰 `04/05/06`

而 `story_event_scheduler.gd` 仍然会用地点过滤事件。

所以当前实际上仍然存在“去哪里决定你会遇到谁”这层逻辑，只是它不再以玩家容易理解的方式呈现。

这会造成两种体验问题：

- 玩家并不知道自己选了某个行动，其实是在“靠近王麻子线”或“靠近疯长老线”
- 当某条线没出来时，玩家不知道是优先级问题、地点问题，还是条件问题

## 四、当前各条线的故事状态

### 柳飞霞线

当前状态：

- 是最完整的一条线
- 有明确起点，有首战，有战后收束，有后续接触
- 已经具备“试探 -> 心战 -> 利用/互相防备”的轮廓

缺口：

- `2002` 和 `2102` 之间还缺更明确的中间升级
- 还缺玩家能主动借她反制王麻子、反向试探她、制造错判的第二层分叉

判断：

- 这条线可以当 Act 1 当前参考标准

### 王麻子线

当前状态：

- 有第 2 天主线 `1002` 做压迫开场
- 有 `2004 -> 9201 -> 9202/9203` 这一套心战入口
- 有“药房旧账、缺页、灭口、筛人”的主题

缺口：

- 玩家不知道为什么“现在该去碰他”
- `2004` 正文承接太弱
- 缺少“从公开压迫转到暗中灭口”的持续升级节点
- 缺少心战胜负之后的持续后果节点

判断：

- 这条线的冲突核心是成立的，但目前只像一个单点事件，不像一条路线

### 疯长老线

当前状态：

- 有 `1004` 作为强力主线锚点
- 有 `2003 -> 9401 -> 9402/9403`
- 有 `2007`
- 有“试药、庇护、交易、把人当材料”的明确危险感

缺口：

- `2003` 入口太空，玩家不清楚为何要主动靠近他
- `2007` 也更像补一刀的提示，而不是持续关系推进
- 缺少“你为了活命必须继续利用他，但每次接近都更危险”的连续交易节点

判断：

- 这是最适合做高风险高回报路线的一条线，但目前只搭了门槛，没有搭走廊

### 夜巡线

当前状态：

- 有 `2005`
- 有 `2008`
- 有 `9301 -> 9302/9303`

缺口：

- 它和王麻子线、化骨池异常线之间还没真正串起来
- 玩家会感到夜巡只是“盘问你的人”，而不是“知道某些东西、代表秩序压力的活节点”

判断：

- 这条线更像辅线，但它应该承担“秩序压力”和“异常目击”的桥接职责

### 化骨池异常线

当前状态：

- 有 `conditional_whisper_deepens`
- 有 `act1_ano_*`
- 有若干夜间风险事件

缺口：

- 还没有形成递进结构
- 还没和“天魔身份感”“族兄之死”“外门筛选”真正合流
- 很多事件仍然只是在提供氛围，不在推动剧情判断

判断：

- 目前这条线像背景噪声，不像主线暗流

## 五、为什么当前玩家不知道怎么开王麻子线和疯长老线

可以直接归纳成 4 个原因：

1. 入口条件藏在底层地点里，但玩家已经看不到地点语义
2. 柳飞霞线优先级更高，先把注意力和事件时机吃掉了
3. 王麻子线、疯长老线入口正文太空，缺少“为什么现在碰它”的理由
4. 中段缺少明确告诉玩家“三条活路分别通向什么”的桥接节点

所以当前问题并不是“再补几场战斗就会好”，而是：

中段缺少“路线声明”和“路线递进”。

## 六、要让故事逻辑完整，最少需要补哪些内容

下面按优先级列。

### 第一批：补主线桥接节点

这是最先要补的。

建议编号段：`1101-1199`

至少需要补 4 类桥：

- `1101` 在 `1002` 后明确三条活路已经浮出水面
  内容目标：告诉玩家，柳飞霞是暗线入口，王麻子是压迫源，疯长老是高风险交易口
- `1102` 在 `1003` 后明确“调查不再是散查，而是在选边”
  内容目标：把柳飞霞、药房旧账、化骨池、试药风险串成同一张网
- `1103` 在 `1004` 后明确“靠近疯长老不是跑题，而是生路之一”
  内容目标：解释为何试药线与月考生存线相连
- `1104` 在 `1005` 前明确“前面拿到的线索该如何收束成路线”
  内容目标：让第六天不再像突然收卷

如果这批不补，后面任何支线补再多，整体仍然会像碎片化节点。

### 第二批：补“路线声明节点”

这是专门解决“玩家不知道怎么开线”的。

建议每条关键路线至少补 1 个显式声明节点。

建议：

- 王麻子线补 `2401`
  内容目标：明确他最近为什么更急、更狠、更需要找替死鬼
- 疯长老线补 `2301`
  内容目标：明确他为什么开始主动留意你，且接近他能换来什么
- 夜巡线补 `2501`
  内容目标：明确夜巡不是普通盘问，而是正在确认某些人是否该被清理
- 化骨池异常线补 `1301`
  内容目标：明确异常不是怪谈背景，而是和族兄之死、筛人机制、天魔身份有关

这批节点的任务不是给大量分支，而是把“路线意义”说清楚。

### 第三批：补每条线的持续推进节点

当前每条线都更像“首轮接触 + 一次战斗 + 一次收束”。
要让它们变成真正路线，至少要补成“入口 -> 升级 -> 后果 -> 再升级”。

建议最少补法：

- 柳飞霞线：补 2 个第二层分叉节点
  主题：互相利用、反向试探、假意同盟
- 王麻子线：补 3 个持续压迫节点
  主题：勒索、补账、借人立规矩、灭口前兆
- 疯长老线：补 3 个交易节点
  主题：试药代价、庇护交换、把你留作材料还是工具
- 夜巡线：补 2 个秩序节点
  主题：盘问、名册、夜巡路线、谁被重点盯上
- 化骨池线：补 3 个递进异常节点
  主题：白天残痕、夜里异声、错名呼唤、污染回响

### 第四批：重做随机事件的职责划分

随机事件不用先大量增加数量，先明确职责。

建议改成 4 类明确用途：

- `资源波动`
  只处理血气、灵石、神识、污染、暴露等
- `人物侧写`
  用来反复强化柳飞霞、王麻子、疯长老、夜巡在别人眼中的样子
- `异常回响`
  只服务化骨池、天魔、族兄死亡回声
- `杂兵磨练`
  只服务心战经验、牌库积累、外门压迫的日常化

随机事件不应该再承担“首次揭露关键设定”的职责。
关键揭露必须回到主线桥接节点或条件节点。

### 第五批：把入口事件正文补齐

这批工作看起来最小，但其实非常关键。

当前至少需要优先补齐这些节点的正文：

- `2003`
- `2004`
- `2005`
- `2007`
- `2008`
- `2102`

原因很简单：

如果入口事件正文是空的，就算系统调度正确、战斗接上了、奖励也发了，玩家仍然只会感到“跳到了下一场事件”，不会感到“进入了一条路线”。

## 七、建议的中段故事完整形态

如果只看故事逻辑，一个比较清楚的中段应该长成这样：

### 第 1-2 天

- 柳飞霞试探你
- 你第一次在心战中压她或被她压住
- 王麻子公开施压
- 玩家意识到“外门秩序本身就是筛人机制”

### 第 2-4 天

- 柳飞霞开始递暗线或继续观望
- 王麻子线开始暴露药房旧账、灭口和替罪结构
- 夜巡线开始提供名册、盘问和异常目击
- 玩家第一次真正意识到化骨池和族兄死因不是分开的

### 第 4-5 天

- 疯长老正式变成可交易的危险路线
- 王麻子和夜巡的压力不再只是资源压迫，而开始转向“谁该被先处理”
- 化骨池异常开始贴近玩家自身，不再只是夜里听到怪声

### 第 5-6 天

- 玩家需要把前面经营的路线收束成一条真正可走的生路
- 柳飞霞、王麻子、疯长老至少要形成一组明确三角关系
- 月考前夜要能回答“你现在依附谁、利用谁、规避谁”

这样第 7 天的结局判定才会显得是“前面选择的自然结果”，而不是骨架突然收束。

## 八、最小补完清单

如果只以“尽快让 Act 1 中段故事完整”为目标，我建议先做下面这份最小包：

1. 补 `1101-1104` 四个主线桥接节点
2. 补 `2003/2004/2005/2007/2008/2102` 的正文
3. 给王麻子线补 `2401-2403`
4. 给疯长老线补 `2301-2303`
5. 给化骨池异常线补 `1301-1303`
6. 重写一批随机事件，让它们按“资源 / 人物侧写 / 异常回响 / 杂兵磨练”分工

如果这 6 步落地，当前 Act 1 的故事逻辑就能从“骨架已在，但中段发虚”，进入“路线清楚、压力清楚、推进有收束”的状态。

## 九、结论

当前中段剧情不清晰，根因不是缺系统，而是缺两样东西：

- 主线桥接
- 路线声明与持续推进

现在玩家不是完全没有内容，而是：

- 柳飞霞线相对完整
- 王麻子线、疯长老线、夜巡线、化骨池异常线都有骨头
- 但这些骨头之间还没有被足够明确地连起来

所以下一步最正确的方向，不是继续加新玩法，也不是继续堆随机事件数量，而是先把中段剧情做成“玩家能看懂自己正在走哪条路”的形态。
