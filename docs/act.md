一、 游戏背景故事
在修仙界最底层、最血腥的【初圣魔门】外门，人命如草芥，杂役皆药渣。
宿主本是一个资质平庸、刚刚失去了族兄庇护的外门杂役，在绝望与恐惧中迎来了自己的死局。然而，宿主濒死前爆发的极度怨念，撕裂了维度的界限，引来了你——玩家【域外天魔】。
你寄生在宿主体内，以凡人的七情六欲、贪嗔痴恨为食。宿主的肉身是你干涉现实的“容器”，而你的能力，是通过在NPC脑海中植入**【魔念】**，扭曲他们的认知，诱导他们自相残杀、吐露秘密，从而吸取他们的“灵魂能量”（经验值），最终反客为主，吞噬整个修仙界。

二、 第一章节剧情大纲：《魔念初醒》
1. 第零世（必死的新手教程 / Tutorial）
场景导入： 阴暗的茅草屋，主角正在为离奇暴毙的族兄烧纸。文字渲染极度的压抑感和无力感。

剧情触发： 柳师姐（柳飞霞）登场。她看似柔弱悲伤，递上疗伤丹药，询问族兄遗物。

玩家交互： 此时 UI 界面只有普通的文字选项（如：隐瞒、交出遗物）。

必死结局： 无论玩家选什么，柳师姐都会露出狰狞的真面目。

选隐瞒：被毒杀。

选交出：被灭口。

表现形式： 屏幕瞬间变红，打出刺眼的文本：“你感觉肝肠寸断/喉咙一甜，你死了。这就是魔门蝼蚁的宿命。”

2. 觉醒时刻 (The Awakening)
UI 突变： 鲜红的死亡屏幕突然碎裂，浮现出暗紫色的“天魔降临”UI 面板。

系统提示音/文本： > “检测到极度怨念，维度锚点已锁定。”

“域外天魔绑定成功。肉身容器重置中……”
“解锁核心天赋：【魔念植入】。看透虚妄，操纵人心。你，才是他们真正的天劫。”


3. 第一世及后续（破局与反客为主）
重返原点： 场景退回茅草屋，柳师姐再次推门而入，和柳师姐对话，她说着一模一样的台词。

全新 UI 面板出现： 此时玩家的界面多出了【天魔感知】（原观察）和【植入魔念】（原入侵）的特殊选项。

破局战： 玩家利用游戏机制，在柳师姐脑中植入对应魔念。成功引动后，柳师姐不仅没有杀人，反而惊恐地吐露了她与执事王麻子的阴谋，并匆匆逃离。

经验结算： 成功扭曲NPC命运，获得【天魔经验】。宿主的修为也随之提升。

转守为攻： 存活下来后，玩家开始在外门游荡，寻找其他精神薄弱的 NPC（如欺软怕硬的杂役、心怀鬼胎的管事），不断“打猎”植入魔念，从被动挨打的猎物，变成潜伏在暗处的精神捕食者。

三、 玩家游戏行动指南 (Player Handbook)
这是玩家在游戏过程中的核心行为逻辑（可以写在游戏的“系统帮助”模块中）：

观察与侧写： 永远不要相信 NPC 嘴里说的话。仔细阅读文本中的动作描写（如“眼神闪躲”、“指尖发抖”），推测他们隐藏的心理破绽。

精准植入： 每次对话只能使用一次【魔念】。如果魔念类型克制了 NPC 的破绽（比如用“贪婪”对付“贪财”的人），会触发【心防击破】，获得经验值和通向好结局的标签。如果植入错误，会导致游戏失败或者通往坏结局。
