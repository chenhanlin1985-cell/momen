## 1. 核心设计理念 (Design Philosophy)
本章的主旨是建立极致的生存压迫感。玩家不走固定的线性剧情，而是处于一个基于条件的“事件池”中。所有事件的发生和选项的解锁，均依赖于玩家的【基础属性】（气血、修为、暴露度）以及后台记录的【全局标记】（Global Flags）。

## 2. 第一章全局标记字典 (Global Flags Registry)
在游戏引擎的 `game_core` 中，需初始化以下布尔值或整型变量，用于记录玩家的轨迹：
- `has_cousin_relic` (bool): 默认 true。主角开局自带的神秘黑色玉简（族兄遗物）。
- `knows_cousin_secret` (bool): 默认 false。是否已向外界暴露族兄遗物的存在。
- `feixia_trust` (int): 默认 0。柳飞霞的信任度/好感度。
- `wang_extortion_paid` (bool): 默认 false。本月是否向王执事缴纳了保护费。
- `wang_anger` (int): 默认 0。王执事的仇恨值。
- `survived_pill_test` (bool): 默认 false。是否成功挺过了疯长老的试药。
- `learned_blood_stealth` (bool): 默认 false。是否掌握了隐藏修为的敛息术。

## 3. 核心 NPC 设定 (NPC Profiles)

### NPC 01: 柳飞霞 (Liu Feixia) - 悬疑/引导位
* **身份背景：** 主角族兄生前的道侣。族兄不久前在宗门内离奇暴毙，死状凄惨。她主动接近孤立无援的主角，看似哀悼旧情、同情主角的遭遇，实则暗藏祸心，试图通过主角找寻那件导致族兄猝死的关键物品。
* **性格标签：** `[多疑]`、`[精通毒理]`、`[极度利己]`
* **交互机制：** 她是高风险高回报的合作者。如果玩家轻信她，可能会被直接毒杀；如果玩家展现出利用价值并手握筹码（玉简），她会成为玩家暗算其他同门的最强利刃。

### NPC 02: 执事王麻子 (Deacon Wang) - 资源消耗/高压位
* **身份背景：** 掌管外门杂役区的中层管事，背后有内门长老做靠山。极其贪婪，视底层杂役为“药渣”和提款机。
* **性格标签：** `[贪得无厌]`、`[欺软怕硬]`、`[心胸狭隘]`
* **交互机制：** 定期触发的“资源税”。玩家必须在“失去资源（灵石/气血）”或“增加暴露度（暴露实力反抗）”之间做出抉择。他是本章最终收束卡点（大考）的主要裁决者之一。

### NPC 03: 炼丹房疯长老 (Mad Elder) - 机遇/死亡位
* **身份背景：** 宗门高层，常年用活人炼制血丹，精神极度不稳定。
* **性格标签：** `[混乱邪恶]`、`[实力碾压]`、`[喜怒无常]`
* **交互机制：** 玩家“暴露度”过高时的惩罚机制。一旦被他盯上，只有极高的“气血”或特定的“标记”才能存活，但活下来后往往能获得越阶的修为提升或特殊身份。

## 4. 自由探索事件池 (Spoke Nodes)
在第 1 到第 9 回合期间，引擎根据 `req_flags` (前置条件) 和权重随机抽取以下事件。

### 事件 A：【深夜的祭奠】
* **触发条件：** 前 3 回合必定触发一次。
* **文本描述：** 柳飞霞提着一篮带着异香的纸钱来到你的茅草屋前，眼眶微红。她递给你一瓶疗伤丹药，叹息着问起你族兄生前是否有交待过什么后事。
* **选项 1：** "收下丹药，矢口否认。" 
  * 效果：气血 +20。`feixia_trust -1`。
* **选项 2：** `[需要 flag: has_cousin_relic]` "拿出黑色玉简，试探她的反应。"
  * 效果：解锁隐藏对话，得知玉简与内门某项禁术有关。设置 `knows_cousin_secret = true`，暴露度 +20，`feixia_trust +5`。

### 事件 B：【执事的剥削】
* **触发条件：** 每隔 3 回合高概率触发。
* **文本描述：** 王麻子带着几个狗腿子踹开了你的房门，冷笑着让你上交本月的“平安例钱”。
* **选项 1：** "咬牙上交 50 块下品灵石。"
  * 效果：灵石 -50。设置 `wang_extortion_paid = true`，暴露度 -5。
* **选项 2：** "囊中羞涩，任其打骂。"
  * 效果：气血 -40，修为跌落 5 点。王麻子骂骂咧咧地离开。
* **选项 3：** `[需要 修为 >= 30]` "暗中运转真气，震断他伸过来的手腕。"
  * 效果：王麻子吃痛退走，但眼神怨毒。设置 `wang_anger +10`，暴露度 +30。

### 事件 C：【丹房的惨叫】
* **触发条件：** `暴露度 >= 50` 时极高概率触发。
* **文本描述：** 你因为最近风头太盛，被疯长老的铁链直接锁进了炼丹房。他要把你当成新版幽冥丹的试药炉鼎。
* **选项 1：** "绝望吞下丹药。"
  * 效果：如果 `气血 < 80`，当场爆体而亡，游戏结束。如果 `气血 >= 80`，扣除 70 气血，修为暴增 50 点，设置 `survived_pill_test = true`。
* **选项 2：** `[需要 flag: knows_cousin_secret]` "大喊：我知道黑色玉简的秘密！"
  * 效果：疯长老停下了手。你用秘密换取了活命的机会，但彻底卷入了宗门高层的旋涡。设置 `survived_pill_test = true`，暴露度直接拉满（99）。

## 5. 章节收束卡点 (Hub Node: 考核危机)
* **触发条件：** `回合数 (Turn) == 10`。此事件强制执行，清算所有 Flags。
* **文本描述：** 外门月末血考。广场上煞气冲天，王麻子坐在高台上，冷酷地翻阅着名册。不合格者，将被直接扔进化骨池。
* **结局分支判定逻辑 (按顺序检查)：**

  1. **【暗潮反杀】(隐秘路线)**
     * 前置条件：`feixia_trust >= 4` AND `knows_cousin_secret == true` AND `wang_anger >= 10`
     * 结算：柳飞霞在暗中配合你，用无色无味的奇毒引动了王麻子体内的旧伤。你趁机发难，将其当场击毙并伪装成走火入魔。你搜刮了他的储物袋，成功晋级。
     * 奖励：进入第二章，获得大量灵石和高级功法。
     
  2. **【长老庇护】(高维碾压路线)**
     * 前置条件：`survived_pill_test == true`
     * 结算：正当王麻子准备刁难你时，疯长老狂笑着降临，一巴掌拍碎了王麻子的天灵盖，宣布你从此是他丹房的专属“药材”，谁也不准动。
     * 奖励：跳过外门，直接以“记名弟子”身份进入第二章内门剧情。

  3. **【苟全性命】(常规路线)**
     * 前置条件：`wang_extortion_paid == true` AND `修为 >= 40`
     * 结算：你战战兢兢地通过了测验，王麻子虽然看你不顺眼，但也没找到发作的借口。你活了下来，继续在外门挣扎。
     * 奖励：进入第二章常规流程。

  4. **【化骨之痛】(死亡结局)**
     * 前置条件：以上均不满足。
     * 结算：你未能满足魔门的残酷法则。王麻子以“修行懈怠”为由，一掌击碎你的丹田，命人将你扔进了化骨池。
     * 奖励：GAME OVER。

## 8. 当前规则约束 (Current Rules)

目前结局语义以运行实现为准：

- `survived_pill_test`：表示你在 day5 活过了疯长老这一关。
- `accepted_elder_patronage`：表示你在 day6 明确选择投靠疯长老。
- 因此：
  - 长老结局不再仅由 `survived_pill_test` 触发。
  - 苟活结局不会再被“只是活过试药”无脑覆盖。
