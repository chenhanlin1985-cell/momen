# 盲玩视角中段叙事审计
## 目的
这份文档不从作者视角看“我们想表达什么”，而是从一个完全没看过设定、也不知道内部状态机的新玩家视角，看当前版本中段故事会如何被实际感知。

重点只回答 4 个问题：
1. 玩家大概率会按什么顺序理解这几条线
2. 哪些地方会觉得信息突然、关系跳跃、动机不明
3. 哪些问题是文本问题，哪些是调度和结构问题
4. 还需要补什么，才能让中段真正达到“第一次玩也能看懂自己在走哪条线”

## 一、盲玩玩家最先感知到的真实结构
如果玩家什么都不知道，只跟着当前版本玩，最先建立起来的认知并不是“四条并行路线”，而是下面这个顺序：

1. 开场已经说明你重开、手里有族兄遗物、眼下最危险的人有柳飞霞和王麻子
2. 第一个真正强烈的可玩节点是柳飞霞试探和首场心战
3. 战后柳飞霞线又最容易继续接上 `2002`
4. 第二天早晨王麻子公开施压，玩家第一次明确感到“外门秩序就是筛人机制”
5. 第二夜 `1101` 才第一次正式把账册、疯长老、化骨池和暂避锋芒列成路线
6. 第三天起，玩家才可能逐步撞见王麻子线、夜巡线、化骨池线、疯长老线的持续节点

所以从玩家体感上说：
- Act 1 现在已经不再是“完全散乱”
- 但它仍然会先被理解成“柳飞霞主线 + 其他几条后来才慢慢浮上来的线”

这不是坏事。问题在于，当前中段还没有完全把“后来浮上来的几条线”稳定接成玩家心里的路线图。

## 二、按盲玩轨迹模拟后的 4 个主要困惑
### 1. 玩家会先以为柳飞霞才是唯一真正的主线
原因不是她写得太多，而是她目前具备最完整的连续体验：
- `2001 -> 9101 -> 9102/9103`
- `2002`
- `2201`
- `2202 / 2203`
- `2102`

这条线有：
- 初见
- 心战
- 暗线建立
- 反试探
- 放假风声
- 持续情报回馈

而其他线虽然已经补到“入口 + 两三拍推进”，但盲玩时还不一定能稳定按顺序撞见。玩家于是很容易先得出一个错误结论：

“这个章节其实就是柳飞霞线，王麻子、疯长老、夜巡、化骨池只是陪衬。”

这会让后面 `1005` 的路线收束显得有点突然，因为系统想让玩家在第六夜收四条线，但玩家心里可能只真正经营了柳飞霞和其中一条副线。

### 2. 王麻子线和疯长老线仍然容易让人觉得“突然进入”
虽然现在 `2004/2401/2402/2403` 和 `2003/2301/2302/2303` 已经比之前完整很多，但盲玩时还有一个感知问题：

- 玩家知道王麻子危险
- 玩家知道疯长老危险
- 但不一定总能明确知道“为什么是今天、为什么这一拍、为什么我现在会撞上这条推进”

也就是说，现在问题已经不是“线不存在”，而是“推进理由还不够稳定可感”。

盲玩时玩家更容易这样理解：
- 王麻子：他又冒出来压我一下
- 疯长老：他又冒出来说一句疯话或谈条件

而不是：
- 王麻子：他因为账册线被逼近，所以正在补洞、立威、准备灭口
- 疯长老：他因为筛人开始进入后半段，所以开始把你从观察对象转成可交易材料

这里缺的不是单个节点，而是更强的“上一拍导致下一拍”的可见承接。

### 3. 化骨池线最容易被玩家当成氛围噪音，而不是主暗流
现在 `1301/1302/1303` 已经把化骨池从纯怪谈拉成了渐进式异常：
- 白天残痕
- 夜里错名
- 标记贴身

这已经比之前好很多。

但盲玩时它仍然有一个天然弱点：
- 柳飞霞线给人的是“有人在试探我”
- 王麻子线给人的是“有人在压我”
- 疯长老线给人的是“有人在拿命和我谈”
- 化骨池线给人的还是“有异常在逼近我”

前三条线都是强人物冲突，最后一条还是偏环境和征兆。

所以玩家容易把它理解成：
- 很有氛围
- 也很重要
- 但暂时不知道它究竟会把我推向谁、改变哪条人线、还是如何进入最终抉择

也就是说，化骨池线现在已经有了“恐怖递进”，但还缺“叙事抓手”。

### 4. 夜巡线最容易被理解成执法阻碍，而不是关键执行层
夜巡线目前有：
- `2005`
- `2008`
- `9301 -> 9302 / 9303`

它已经承担了一部分“秩序在执行”的感觉，但盲玩时还有一个问题：

玩家很容易把夜巡弟子看成：
- 来问你话的人
- 来确认你知不知道什么的人
- 来增加压力的人

但不一定会立刻把他理解成：
- 王麻子补洞节奏的外层执行者
- 外门筛人秩序的基层接口
- 化骨池异常和失踪传闻的实际目击层

所以夜巡线在玩家脑中的地位，可能还是“压力型支线”，而不是“让整个筛人机制落地的关键视角”。

## 三、比文本更伤体验的两个结构问题
### 1. 新补节点的调度优先级基本是空的
这是当前最大的不稳定因素之一。

像：
- `2201-2203`
- `2301-2303`
- `2401-2403`
- `1301-1303`

这些新补节点在 [events.csv](/C:/momen/content/story/act1/csv/events.csv) 里现在的 `schedule_priority` 基本是空的。

从盲玩角度，这会带来一个非常危险的问题：
- 节点写得再清楚，也未必能按作者预期顺序被撞见
- 玩家可能先看到后一拍，再看到前一拍
- 或者被柳飞霞高优先级节点长期压住

如果这一层不收，玩家感受到的“突兀”有一部分根本不是文案问题，而是调度顺序问题。

### 2. 部分路线节点仍然更像“碰到就触发”，不像“被前文推出来”
现在很多中段节点触发条件还是偏轻：
- `day_gte`
- `phase_is`
- 某条 `route_*`
- 个别状态存在

这在系统上足够，但在盲玩体验上会导致：
- 玩家不知道自己是因为什么触发了这拍
- 也不知道错过这一拍会不会损失什么
- 更不知道这拍究竟是路线推进、气氛补写，还是一次普通波动

从玩家视角，这种“看上去像随机冒出来”的感觉，就是突兀的来源。

## 四、从玩家视角最突兀的几个点
### 1. `1003` 到 `1101/1102` 之间仍然有轻微跳轴感
玩家会记住：
- 王麻子压迫
- 柳飞霞试探
- 第一次正式选边

但在真正形成“路线图”之前，玩家还需要一个更直接的心理转换：

“我已经不是在东撞西撞了，我现在真的在经营路线。”

`1101` 和 `1102` 已经在做这件事，但盲玩时仍可能稍显晚了一拍。因为玩家在此之前已经打完柳飞霞、见过王麻子，脑中可能已经偷偷形成了自己的主线判断。

### 2. `2102` 现在更像有价值的情报回馈，但还不是强分叉回报
柳飞霞线现在已经顺得多了，但 `2102` 对玩家来说，仍然更像：
- 她继续给我送消息

而不是：
- 她因为我之前的经营方式不同，已经变成了不同类型的人

换句话说，柳飞霞已经有分叉的形，但不同分支的“人物状态差异感”还可以更强。

### 3. 疯长老线有强概念，但少一次真正的“非他不可”证明
疯长老现在的节点已经很像一路高风险交易，但盲玩时玩家还是可能会问自己：

“我为什么一定要走这条？”

王麻子线能提供把柄，柳飞霞线能提供情报，化骨池线能提供真相残痕。
疯长老线提供的是“活着被留下”的资格，可这份价值目前更多是作者知道，玩家还没有被一次足够扎实的后果拍脸。

也就是说，疯长老线现在已经讲清“危险”，但“必要性”还不够强。

### 4. 化骨池线还缺一次明确把它和某个人线咬死的节点
现在它和：
- 族兄之死
- 外门筛选
- 天魔身份

都已经有关系。

但盲玩玩家仍可能会问：
- 化骨池到底主要咬的是王麻子？
- 还是夜巡？
- 还是疯长老？
- 还是它本身就是一条独立真相线？

也就是：
它现在是“重要”，但还没有“落到谁身上最痛”的那一下。

## 五、按盲玩体验排序，最该补的不是更多节点，而是这 5 类承接
### 1. 给每条线补一条“上一拍为什么把你推到这里”的短承接
不是再加大段背景，而是让关键节点开头更明确回答：
- 我为什么今天会来这里
- 是前一拍哪件事把我推过来的

这类承接最该补在：
- `2004`
- `2005`
- `2003`
- `2301`
- `2401`
- `1301`

### 2. 给柳飞霞线补“不同经营方式导致不同人物状态”的更强回显
当前柳飞霞线已经能走通，但玩家还不一定强烈感觉到：
- 她是被你经营成线人
- 被你逼成短期同盟
- 还是被你压成恐惧驱动的危险工具

下一步应补的不是更多“她又来报信”，而是更强的分支人格回显。

### 3. 给疯长老线补一次“没有这条线就真会死得更快”的实证
疯长老线现在的概念价值足够，但玩家还缺一次真正感到：
- 若没有这条线，自己在筛人机制里会更快被刷掉

这可以通过：
- 一次试药名单前兆
- 一次被记上可弃名单的暗示
- 一次其他路线无法解决的生存困局

来实现。

### 4. 给化骨池线补一个“明确咬住某条人线”的中间节点
最需要的是一次明确答案：
- 化骨池残痕最终牵回的是谁的手
- 或至少先牵回谁负责补口、谁负责巡路、谁负责递话

这样玩家才会把它从“重要氛围”升级成“能推动判断的证据线”。

### 5. 给夜巡线补一次“执行面真相”的揭示
夜巡现在最缺的不是更多盘问，而是一次明确让玩家理解：
- 夜巡不是单纯来盯我
- 夜巡是在替筛人机制做底层确认
- 他们自己也知道某些失踪和异常，只是不敢直接说

一旦这层补上，夜巡线的地位会从“压力支线”升成“秩序实务层”。

## 六、应该先做的具体内容
如果只按“让第一次玩的玩家不迷糊”来排优先级，建议顺序如下：

1. 先补一份“盲玩路线图”式内容表，列出第 1 到第 6 天玩家大概率获得的关键信息
2. 给 `2201-2203 / 2301-2303 / 2401-2403 / 1301-1303` 手动补调度优先级
3. 给 `2003 / 2004 / 2005` 的开头再加一层“上一拍推动理由”
4. 给疯长老线补一个“非他不可”的实证节点
5. 给化骨池线补一个“它最终咬住谁”的人物化节点
6. 给夜巡线补一个“他们在执行筛人，不只是盘问你”的显性节点

## 七、结论
当前版本已经比之前充实很多，问题不再是“中段没有内容”，而是：

- 玩家已经能感觉到几条线存在
- 但还不总能稳定理解“自己为什么此刻在走这条”
- 也还不总能把这些线在脑中收成一张清晰的路线图

从盲玩视角看，当前最需要优先解决的并不是再继续横向加很多新节点，而是：

- 稳定调度顺序
- 强化上一拍到下一拍的承接
- 让每条线更明确地回答“它到底解决什么问题”

只要这三件事做稳，中段就会从“内容很多但偶尔突兀”，真正进入“第一次玩也能顺着理解”的状态。
