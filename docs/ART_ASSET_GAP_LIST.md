本文档只统计当前版本真实会影响运行时体验的美术资源缺口，并直接给出可用于 AI 生成的提示词。

约束：
- 资源命名继续遵守当前项目数字目录规范
- 本文档只列当前版本已经有运行时入口的资源
- 提示词以统一世界观为目标：低阶修真门派外门、压抑、潮湿、贫瘠、危险、偏写实国风插画
- 所有提示词默认用于生成 2D 游戏立绘或横版背景，不要生成 3D、Q 版、现代服饰、赛博元素

## 六、给 AI 出图时的统一附加要求

每次生成时都建议在主提示词后追加这些约束，尽量统一风格：

```text
二维游戏插画，手绘风格，国风黑暗奇幻，人物比例写实，不要现代物件，不要赛博元素，不要多余手指，不要水印，不要文字，不要 logo，轮廓清晰，适合视觉小说和叙事 RPG 的 UI 覆盖
```

如果是做背景，再追加：

```text
横版构图，给 UI 留出负空间，前景不要近景人物
```

如果是做立绘，再追加：

```text
半身角色立绘，构图居中，透明或简洁纯色背景，在游戏 UI 缩放下表情仍然清晰可读
```
