# [事件] 柳飞霞递来第一份暗报
@编号: 2102
@剧情分类: 条件剧情事件
@内容分类: npc_state
@表现形式: 三段式对话事件
@触发阶段: 行动后
@触发时间: 白天
@地点: 茅草屋
@条件: 第 3 天及以后, global.liu_informant_active == true

[开场]
第三天白日，柳飞霞没有敲门，只在门缝里塞进来一张折了三次的薄纸。等你抬头时，她已经站在屋檐阴影里，像一只既想靠近又随时准备逃走的鸟。她低声说，王麻子这两天改了巡路和收钱的时辰，今晚还会去一趟药房后墙。


[开场表情: 悲伤]
[观察]
表层情绪：紧绷、急切、试探。

[观察表情: 悲伤]
环境细节：她说到“今晚”两个字时舌尖明显发干，像是昨夜根本没睡；可提到“你打算怎么用这消息”时，眼神又会本能地闪一下。她已经踏进了你的局，却还在替自己留能抽身的余地。

[入侵: 贪]
把这次合作描成一桩真正能换活路和好处的交易，让她先盘算你值不值得继续押注。

[入侵提示: 贪]
她的慌张没有消失，只是迅速换成了算账。你能感觉到她不是不怕死，而是在想：这份消息能不能让她换到比恐惧更稳的退路。


[入侵表情: 贪 = 生气]
[入侵记录: 贪]
你把“活路有价”这件事钉进了她最擅长盘算的一层心思里。

[入侵: 嗔]
继续放大她对王麻子的惧意，让她相信今晚若不先把刀递给你，明天被灭口的人就会是她。

[入侵提示: 嗔]
一提到王麻子，她的肩背就会先绷紧，再硬生生压下去。她现在最怕的不是与你翻脸，而是自己迟一步就会变成替罪羊。


[入侵表情: 嗔 = 生气]
[入侵记录: 嗔]
你把她不肯承认的杀机重新翻出来，逼她正视“再不选边就来不及”的那一刻。

[入侵: 痴]
让她短暂相信，只有你会把她当成同一条船上的人，不会像王麻子那样把她先推出去挡刀。

[入侵提示: 痴]

[入侵表情: 痴 = 悲伤]
她听到“同路人”时有一瞬松动，可那点松动很快又被戒心裹住。她并非不会动摇，只是还不敢真把背后交给任何人。

[入侵记录: 痴]
你把同舟共济的幻觉塞进她心里，让她第一次认真想象“也许可以不先互相出卖”。

[对话]
=> [先听她把今夜要说的事讲完]
你不急着逼问，只示意她继续往下说。
$线索碎片: +1$

=> [逼问她为什么现在才来递这份消息]
你没有先接那张纸，而是盯着她，问她为何拖到今天才肯把这种事递到你门前。
$获得状态: 柳飞霞存在背叛风险$

[对话: 贪]
=> [顺着她的口风谈条件，先听她想拿这消息换什么]
柳飞霞见你没有立刻追问，反而先低低报出了两个名字和一处碰头时辰。她没再装可怜，而是直接问你：若她继续替你盯着王麻子，你能不能给她一条不被一起灭口的退路。
$获得状态: 已和柳飞霞约定暗报地点, 线索碎片: +1.0$

=> [把价码摆到明面上，逼她开出一个背叛王麻子的条件]
你把“忠诚”和“价码”摆上桌面，柳飞霞眼里的算计反倒稳定下来。她承认自己会继续递消息，但每一步都要换筹码。她没有拒绝你，却也把“随时改价”写在了脸上。
$获得状态: 已和柳飞霞约定暗报地点, 获得状态: 柳飞霞存在背叛风险$

[对话: 嗔]
=> [不打断她，让她自己说出今晚最怕王麻子做什么]
柳飞霞越说越快，最后几乎是在咬牙：王麻子今晚会先去后墙，再转去外门广场收尾，像是在确认谁敢先把消息漏出去。她把时辰、路线和交接口都抖了出来，像是终于承认自己已经站在刀口上。
$获得状态: 已掌握王麻子近期行程, 获得状态: 已和柳飞霞约定暗报地点, 线索碎片: +2.0$

=> [逼她现在就把王麻子的暗线、传话口和今夜路线一起交出来]
你不给她留下继续试探的空间。柳飞霞沉默片刻，终于把纸摊开，连同王麻子的巡路时辰、藏账口和一个专门传话的暗号一起全交到了你手里。她这次给出的，已经不只是消息，而是一把足够伤到王麻子的刀。
$获得状态: 已掌握王麻子近期行程, 获得状态: 已和柳飞霞约定暗报地点, 获得状态: 王麻子暗线已暴露, 线索碎片: +3.0$
