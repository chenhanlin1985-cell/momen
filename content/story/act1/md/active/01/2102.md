# [事件] 柳飞霞递来第一份暗报
@编号: 2102
@剧情分类: 条件剧情事件
@内容分类: npc_state
@表现形式: 三段式对话事件
@触发阶段: 行动后
@触发时间: 白天
@地点: 茅草屋
@条件: 第 3 天及以后, global.liu_informant_active == true

[开场]
第三天白天，柳飞霞没有敲门，只在门缝里塞进来一张折了三次的薄纸。等你抬头时，她已经站在屋檐阴影里，像一只既想靠近又随时准备抽身的鸟。

她低声说，王麻子这两天改了巡路和收钱的时辰，今夜还会去一趟药房后墙。她递来的不只是情报，也是第一次真正把自己往你这边押了半步。

[观察]
表层情绪：紧绷、急切、仍在试探退路。

[观察表情: 悲伤]
环境细节：她说到“今夜”两个字时舌尖明显发干，像是昨夜根本没睡；可一提到“你打算怎么用这消息”，目光又本能地闪了一下。她已经踏进了你的局，却还在替自己留抽身的空位。

[侵入: 贪]
把这次合作描成一桩真能换活路和好处的交易，让她先盘算你值不值得继续押注。

[侵入提示: 贪]
她的慌张没有消失，只是迅速换成了算账。她在想的不是该不该继续说，而是这份消息能不能替她换来一条更稳的退路。

[侵入记录: 贪]
你把“活路有价”这件事钉进了她最擅长盘算的那层心思里。

[侵入: 嗔]
继续放大她对王麻子的惧意，让她相信今夜若不先把刀递给你，明天被灭口的人就会是她。

[侵入提示: 嗔]
一提到王麻子，她的肩背就会先绷紧，再硬生生压下去。她现在最怕的不是与你翻脸，而是自己晚一步就会变成替罪羊。

[侵入记录: 嗔]
你把她不肯承认的杀机重新翻了出来，逼她正视“再不选边就来不及”的那一刻。

[侵入: 痴]
让她短暂相信，只有你会把她当成同一条船上的人，不会像王麻子那样先把她推出去挡刀。

[侵入提示: 痴]
她听到“同路人”时有一瞬松动，可那点松动很快又被戒心裹住。她并非不会动摇，只是还不敢真把背后交给任何人。

[侵入记录: 痴]
你把同舟共济的幻象埋进她心里，让她第一次认真想象“也许可以不先互相出卖”。

[对话]
=> [先听她把今夜要说的事讲完，再看她究竟想把你推向哪条线]
你没有急着追问，只示意她继续往下说。柳飞霞把药房后墙、巡夜换班和王麻子今夜会亲自去看的一处口子都说了出来。到这一步，你已经能确定，她给的不是闲话，而是一条真正能往药房线深处探的入口。
$线索碎片: +1, 天魔经验: +1, 获得状态: 已掌握王麻子近期行程, 获得卡牌: 暗报折纸$

=> [追问她为什么偏偏现在才把这份消息递过来]
你没有先接那张纸，而是盯着她问，为什么拖到今天才肯把这种消息送到你门前。柳飞霞先沉默，最后还是承认她之前一直在等，等王麻子先把刀架到她自己脖子上，才敢确定你不是更快卖掉她的那一个。她把话说得很难听，却也等于把真正站队的理由一起说了出来。
$获得状态: 柳飞霞存在背叛风险, 获得状态: 柳飞霞已转而针对王麻子, 天魔经验: +1, 获得卡牌: 暗报折纸$

[对话: 贪]
=> [顺着她的口风谈条件，先听她想拿这消息换什么]
柳飞霞见你没有立刻追问，反而先低低报出了两个名字和一处碰头时辰。她没有再装可怜，而是直接问你：若她继续替你盯着王麻子，你能不能给她一条不被一起灭口的退路。她这次递来的不只是消息，也是明码标价的投靠。
$获得状态: 已和柳飞霞约定暗报地点, 线索碎片: +1, 天魔经验: +1, 获得卡牌: 暗报折纸$

=> [把价码摆到明面上，逼她开出一份真正的投名状]
你没有给她含糊试探的空间，而是让她现在就决定要拿什么换继续合作。柳飞霞想了很久，最后把王麻子最近换巡路、换收钱时辰、甚至连药房后墙那条暗线都一并交了出来。她没有变得可信，只是先把更值钱的那部分押到了你手里。
$获得状态: 已和柳飞霞约定暗报地点, 获得状态: 王麻子暗线已暴露, 获得状态: 已掌握王麻子近期行程, 线索碎片: +2, 天魔经验: +2, 获得卡牌: 暗报折纸$

[对话: 嗔]
=> [不打断她，让她自己把今夜最怕王麻子做什么说出来]
柳飞霞越说越快，最后几乎是在咬牙：王麻子今晚会先去后墙，再转去广场收尾，像是在确认谁敢先把消息漏出去。她把时辰、路口和交接口全抖了出来，像是终于承认自己已经站在刀口上。到这一刻，她给你的已经不只是消息，而是一把能伤到王麻子的刀。
$获得状态: 已掌握王麻子近期行程, 获得状态: 已和柳飞霞约定暗报地点, 线索碎片: +2, 天魔经验: +1, 获得卡牌: 暗报折纸$

=> [逼她现在就把王麻子的暗线、传话口和今夜路数全交出来]
你不给她留继续摇摆的余地。柳飞霞盯着你看了很久，最后把折纸彻底摊开，连同王麻子的巡路时辰、藏账口和一处专门传话的暗号一起全交了出来。她这次给出的，已经足够把“药房那边有人在补洞”从怀疑推进成可用的把柄。
$获得状态: 已掌握王麻子近期行程, 获得状态: 已和柳飞霞约定暗报地点, 获得状态: 王麻子暗线已暴露, 线索碎片: +3, 天魔经验: +2, 获得卡牌: 暗报折纸$
