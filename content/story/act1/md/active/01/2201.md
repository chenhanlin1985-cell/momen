# [事件] 柳飞霞定下暗报规矩
@编号: 2201
@剧情分类: 条件剧情事件
@内容分类: npc_state
@表现形式: 普通事件
@触发阶段: 行动后
@触发时间: 白天
@地点: 茅草屋
@条件: 第 3 天及以后, global.liu_contact_established == true, global.liu_informant_active == false

[开场]
柳飞霞这次没有直接递消息，而是先把一粒石子轻轻弹进你门槛边的灰里。她站在屋檐阴影下，盯着那粒石子看了一会儿，才低声说，若以后真要继续往来，就不能再像前两次那样临时碰头。

她说王麻子最近已经开始连宿舍口和后墙边的小动静都要过问，谁若总在同一个时辰出现在同一个地方，迟早会被顺着摸出来。她不是来跟你讲义气，而是来定规矩，看看你值不值得她把这条暗线真的接起来。

[观察]
表层情绪：谨慎、试探、仍在给自己留后手。
[观察表情: 悲伤]
环境细节：她说到“规矩”时语速比平时稳，像是一路想过很多遍；可一提“若被摸出来”，手指还是会不自觉按向袖口。她不是不想靠近你，而是不肯先把自己的退路完全交出来。

[侵入: 贪]
让她先按交易的路数往下走，相信一条稳定暗线本身就能替她换来生路。
[侵入提示: 贪]
她的防备没有消失，只是慢慢换成了算账。只要这条暗线够稳、够值钱，她就会愿意自己把它接实。
[侵入记录: 贪]
你把“合作有价”这件事往她最会盘算的地方又按深了一层。

[侵入: 嗔]
让她更清楚地意识到，再不把规矩定下来，王麻子早晚会先把她卖出去顶锅。
[侵入提示: 嗔]
一提到“先被顺着摸出来的那个人”，她眼里那点犹豫就先烧成了狠意。她现在最恨的不是你，而是自己若再慢一步就会先成弃子。
[侵入记录: 嗔]
你把她对王麻子的戒心推成了必须立刻做决定的焦躁。

[侵入: 痴]
让她短暂相信你们已经在同一条船上，只有先把规则定死，彼此才都有活路。
[侵入提示: 痴]
她确实被这句话拉近了一步，但那种拉近更像暂时借你站稳，而不是彻底把背后交给你。
[侵入记录: 痴]
你把同路人的幻象轻轻压进了她的防备里。

[对话]
=> [答应按她的规矩走，先把暗报地点和传话方式定下来]
你没有和她争谁先留后手，只先把能落地的规矩定了下来。柳飞霞说以后若真有急信，不走宿舍门口，也不走白天的正路，只在后墙和破屋之间留一处短停、一处暗号。她说得越细，越说明她这次不是来试你，而是真的把自己往这条线里放了一步。$获得状态: 柳飞霞已成为线人, 获得状态: 已和柳飞霞约定暗报地点, 线索碎片: +1, 天魔经验: +1, NPC关系: 柳飞霞.好感 +1$

=> [不急着全信，先让她把规矩说完，再看她有没有故意留口子]
你没有立刻答应，只把她定下的时辰、暗号和停步位置一条条听完，专门去看她哪里会故意留空。柳飞霞看得出你在防她，却也没有翻脸。她最后还是把大半规矩说清了，只给自己留了最后一点能转身的余地。到这一步，她已经够资格做线人，却也仍旧值得防。$获得状态: 柳飞霞已成为线人, 获得状态: 已和柳飞霞约定暗报地点, 获得状态: 柳飞霞存在背叛风险, 线索碎片: +1, 天魔经验: +1$
