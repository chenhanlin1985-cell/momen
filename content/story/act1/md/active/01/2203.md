# [事件] 柳飞霞替你放出假风声
@编号: 2203
@剧情分类: 条件剧情事件
@内容分类: npc_state
@表现形式: 普通事件
@触发阶段: 行动后
@触发时间: 白天
@地点: 茅草屋
@条件: 第 5 天及以后, global.liu_informant_active == true, global.wang_schedule_exposed == true

[开场]
你已经从柳飞霞那里拿到过几回真消息，可这一次你不想只继续被动接她递来的东西。若她真能用，就该不止会报信，还该能替你往外放一阵你想让别人听见的风。

柳飞霞听懂你意思之后，先是皱眉，像在估量这件事会不会让她死得更快。可她最后没有一口回绝，只问你准备让王麻子听见哪一种版本：是有人已经摸到账缝，还是有人正要往化骨池那边追。

[观察]
表层情绪：犹豫、兴奋、又怕失手。
[观察表情: 悲伤]
环境细节：她怕的不是“骗人”本身，而是骗人之后那一下反噬会先落到谁身上。可你也能看出来，她对这种反手布线并不陌生，甚至在某些瞬间，还有点像终于轮到自己先下子。

[侵入: 贪]
让她更在意这次假风声若放得漂亮，自己就能从被动挨打那边翻回一点主动。
[侵入提示: 贪]
她开始真正盘算这次出手能给自己换来多少身位，不再只把它当成替你跑腿。
[侵入记录: 贪]
你把她对“借假风声翻身”的心动推到了前面。

[侵入: 嗔]
让她把这次放风当成先给王麻子一记回刀，借此逼他自己露出暗线。
[侵入提示: 嗔]
她提起王麻子时那点咬牙的劲重新冒了出来。若能让那个人先乱一步，她愿意冒的险会比平时更大。
[侵入记录: 嗔]
你把她心里那把迟迟没落下去的刀重新举稳了。

[侵入: 痴]
让她暂时相信你已经给她留好了退路，只要风声放出去，就不会先拿她去挡反噬。
[侵入提示: 痴]
她被这句话稳住了一点，可你也知道，这种稳只维持到局面第一次反咬回来之前。
[侵入记录: 痴]
你给了她一个足够行动的错觉，让她肯先替你出手。

[对话]
=> [让她放出“有人已经摸到账缝”的假风声，看王麻子先去堵哪一处]
你没有让她直接递真消息，而是让她故意往外漏一句半真半假的话：已经有人顺着旧账和缺页摸到了王麻子最近补洞的那一层。若他先去堵账册、后墙或旧账室里的某一处，那本身就是在替你指出真正最疼的地方。柳飞霞听完只点了点头，像是已经在心里替这句话找好了该落进谁耳朵里的路。$获得状态: 王麻子暗线已暴露, 线索碎片: +1, 天魔经验: +1$

=> [让她放出“有人往化骨池那边追”的风，看谁会比王麻子先着急]
你换了个更险的版本，叫她把风往化骨池那边吹。若先跳出来的不是王麻子，而是夜巡、后墙口或别的传话口，那就说明池边那条线牵着的手远不止一只。柳飞霞听完没有立刻应，最后还是说她可以试，但这一下若炸响，后面收不收得住就不全由她决定了。$获得状态: 王麻子暗线已暴露, 获得状态: 柳飞霞存在背叛风险, 线索碎片: +1, 天魔经验: +1$

[对话: 嗔]
=> [借她对王麻子的恨，把这次假风声做成逼他自乱阵脚的一刀]
你没有再把话说得含糊，而是直接告诉她，这次就是要逼王麻子先乱，先自己露出补洞的手。柳飞霞沉默片刻，最后笑得很冷，说她知道该把这阵风送去哪几个人耳朵里，既能让他听见，又不至于立刻查到你门前。到了这一步，她已经不只是被你拿来报信，而是真的开始替你出刀。$获得状态: 柳飞霞已转而针对王麻子, 获得状态: 王麻子暗线已暴露, 线索碎片: +2, 天魔经验: +1$
