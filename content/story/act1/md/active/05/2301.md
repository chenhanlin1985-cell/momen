# [事件] 疯长老开价
@编号: 2301
@剧情分类: 条件剧情事件
@内容分类: npc_state
@表现形式: 紧凑抉择事件
@触发阶段: 行动后
@触发时间: 白天
@条件: 第 4 天及以后, global.route_seek_senior == true, global.met_mad_elder == true

前几天疯长老留给你的那几句疯话，原来不是随口吓人。你越顺着这条线去想，就越看清他真正盯着的从来不是谁胆子小、谁嘴硬，而是谁已经被外门那套筛人规矩逼到了边上。

所以这一次他第二次主动叫住你时，话也不再绕了。他站在丹房门口，看着被抬出去又抬回来的药桶，像是随口问你怕不怕试药，下一句却直接落到了价码上。

他说外门里求他的人很多，可真正能被他留下来的，不是最听话的，也不是最能忍的，而是“死了可惜、活着有用”的那一类。若你真想把他这条险路当活路走，就该先想清楚，自己准备拿什么证明“还值得留着”。

=> [先顺着他的价码往下听，看看他到底想留什么样的人]
你没有急着应，也没有退，只顺着他那句“死了可惜”问了下去。疯长老笑得比先前更真了一点，说近来外门里失踪的、被记走的、被临时换进试药名单的，从来都不是三回事。只要有人开始补名册、调夜路、改交接，离正式挑人就不远了。$获得状态: 已确认失踪传闻属实, 线索碎片: +1, 天魔经验: +1$

=> [不先答应卖命，先把这份开价记下，等真要换命时再回来谈]
你没在这一步把自己送出去，只把他的开价一字一句记在心里。疯长老要的不是恭顺，而是能被他拿去换、更能继续拿来用的命。明白这一点之后，你再回头看丹房里那一声声惨叫，就知道自己不是在听怪事，而是在听筛人的规矩。$线索碎片: +1, 天魔经验: +1$
