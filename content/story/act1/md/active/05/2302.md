# [事件] 试药之后的回看
@编号: 2302
@剧情分类: 条件剧情事件
@内容分类: npc_state
@表现形式: 紧凑抉择事件
@触发阶段: 阶段开始时
@触发时间: 晨间
@条件: 第 6 天及以后, global.route_seek_senior == true, global.met_mad_elder == true

你前一天刚把疯长老那句“死了可惜、活着有用”记进心里，第五夜丹房里的惨叫就真的替这句话补上了代价。到了天亮时，整个外门反而安静得过分，像是谁都默认昨夜那一轮筛选不该被拿出来议论。

疯长老没有再专门来找你，可你已经能看出一些先前看不见的东西。

昨夜被叫进丹房的人，今晨有几个照常出现，有几个却像从来没在名册上存在过。活下来的，被悄悄记到另一张单子里；没活下来的，连议论都被压得极轻。到这一步你终于彻底看明白，疯长老从来不是在胡乱试药，他是在替某种更大的筛选挑“还能继续用的人”。

=> [把昨夜活下来的名单和外门筛选联系起来]
你终于把疯长老、试药和外门筛选放进了同一套逻辑里。若他愿意留人，那不是慈悲，而是那个人还没被榨干价值。想到这里，疯长老这条线也就不再只是险路，而是一笔越来越清楚的交易。$获得状态: 已接受疯长老庇护, 线索碎片: +1, 天魔经验: +1, 获得卡牌: 留用药签$

=> [先不急着向他低头，只记住“活下来的人会被另册记走”]
你没有立刻把自己送进那份名单里，只是先记住了这条规矩。只要还没被真正记走，就还有绕开的余地；可一旦真到了非换不可的时候，你也知道该拿什么和疯长老谈了。$线索碎片: +1, 天魔经验: +1, 获得卡牌: 留用药签$
