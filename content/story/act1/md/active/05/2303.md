# [事件] 疯长老索要投名状
@编号: 2303
@剧情分类: 条件剧情事件
@内容分类: npc_state
@表现形式: 紧凑抉择事件
@触发阶段: 行动后
@触发时间: 白天
@条件: 第 6 天及以后, global.route_seek_senior == true, global.met_mad_elder == true

到了第六天，你已经亲眼看见疯长老那套“活下来的人会被另册记走”的规矩是怎么运转的，所以这一次他再开口时，你很清楚，谈的已经不再是要不要靠近，而是靠过去之后你准备拿什么把自己押上去。

离月考只剩最后一轮时，疯长老终于把那层含糊笑意揭开了。他说若你真准备借他的手活下去，就不能只拿几句试探和几份怀疑来换。

他要的不是口头上的靠拢，而是一份投名状。可以是一条别人还不知道的线索，可以是一个会被先筛掉的名字，也可以是你亲口承认，真到要命的时候，你愿意先走进他的规矩里，再谈怎么走出来。到了这一步，疯长老线已经不再是在旁边看看风向，而是在逼你决定，要不要把“被留下来继续用”当成自己的活路。

=> [先交出一份能让他立刻用上的消息，换一个“暂时留着”的位置]
你没有再装局外人，而是把手里能立刻派上用场的那部分先递了出去。疯长老听完没有立刻表态，只是把你从“看看也无妨”的名单，往“先留着观察”的那边挪了一格。到了这一步，你已经不只是被他看见，而是真的把命压到他这条线上了。$获得状态: 已接受疯长老庇护, 线索碎片: +1, 天魔经验: +1$

=> [先拖住他，不把投名状一次交完，给自己留最后一层退路]
你没有在最后一天把底牌交净，只把愿意靠过去的姿态先放了出来。疯长老看得懂你在留后手，却也没有拆穿。对他这种人而言，肯靠近的人已经比嘴上忠心的人更值钱；而你至少替自己保住了最后一层能转身的余地。$线索碎片: +1, 天魔经验: +1$
