# [事件] 后墙私换
@编号: 3401
@剧情分类: 条件剧情事件
@内容分类: random_disturbance
@表现形式: 普通事件
@触发阶段: 行动后
@触发时间: 白天
@条件: 第 5 天及以后

外门里真正值钱的东西，很少会明着摆在台面上卖。到了第五天后，后墙边已经有人开始借着换岗、递桶和跑腿的工夫，悄悄把一些不能见光的东西塞来换去。

你看得出，这不是正经买卖，更像是谁把命和消息都切成了碎片，趁月考前还能换的时候赶紧出手。有人卖的是一条暗线，有人卖的是一口没烧干净的炉灰，还有人只收钱，替你把一张碍手的牌悄悄处理干净。

=> [花两块灵石，换下那张写满价码的血书残页](消耗灵石: 2)
你把灵石塞过去，对方连头都没抬，只把一张带着旧血锈味的残页推到了你手里。上头没有完整名字，只有被反复圈过的价码和改过的时辰，可正因为不完整，才更像一把能顺着恐惧往里撬的薄刀。$获得卡牌: 血书价码$

=> [花两块灵石，换一张从废炉里摸出来的余温药纸](消耗灵石: 2)
那人把纸包递给你时，指尖还有一股被炉灰熏出来的苦味。纸包里夹着的不是完整药方，而是几段被烧剩的边角，可只消看一眼，你就知道它记下的不是治病，而是谁曾被拿去试过、烧过、留过。$获得卡牌: 废炉余温$

=> [花一块灵石，请人把《杂役的日常》这张废牌处理掉](消耗灵石: 1)
你没有再给自己留那张最拖手的庸常废牌，而是让后墙边的人替你把它烧得干干净净。灵石少了一块，牌组却轻了一层，往后再摸牌时，你也少了一次被废话堵手的机会。$移除卡牌: 杂役的日常$

=> [先不碰这摊浑水，把后墙这条私换路记下来]
你没有立刻把灵石花出去，只是把这条私换路和出没时辰先记了下来。外门里越是月考将近，越有人会急着把不能见光的东西往外甩；真到要用的时候，这里迟早还能再来。$线索碎片: +1$
