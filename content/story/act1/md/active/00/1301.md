# [事件] 白天留下的池痕

@表现形式: 紧凑抉择事件

@条件: 第 3 天及以后, global.route_well == true

你前几夜一直被化骨池那边的异声往回拽，可真到了白天再顺着后墙和偏路细看，才发现最吓人的东西反而不是夜里那点声音，而是这些本该被人清干净、却还是压不住的痕。

墙根有被什么东西反复蹭过的白印，石缝里还有干了又返潮的黑灰，像是有人曾被拖到边上，又有人急着把留下来的东西刮掉。

到了这一步，化骨池已经不是怪谈，而是一处白天也会往外渗线索的地方。夜里的异声不过是在催你回头看白天这些本来就存在的残痕。

=> [承认这不是怪谈，而是有人被带到这里后留下的真痕]
你不再把那些异声和白印拆开看了。白天留下来的痕，反而比夜里的声音更像证据。只要有人来过、被拖过、被处理过，那化骨池就和外门里最近这些失踪与遮掩脱不了干系。$获得状态: 已确认失踪传闻属实, 线索碎片: +1, 天魔经验: +1$

=> [拿族兄留下的碎线去对这些痕，先确认他是不是也来过这里]
你把族兄留下来的只言片语和眼前这些残痕一一对上，越对越觉得发冷。若他不是死在化骨池边，至少也一定被带着路过了这处地方。你之前那些关于族兄之死的怀疑，到这一步终于有了可以落脚的地方。$获得状态: 已确认族兄之死有异, 线索碎片: +1, 天魔经验: +1$
