# [事件] 旧账室里的人名
@编号: 2401
@剧情分类: 条件剧情事件
@内容分类: npc_state
@表现形式: 紧凑抉择事件
@触发阶段: 行动后
@触发时间: 白天
@条件: 第 3 天及以后, global.route_records == true

第三天一早你已经想明白，若真要顺着账册这条线往里翻，盯的就不能只是王麻子脸上的那点慌。真正会留下证据的，是他来不及全补平的旧账和旧名。

药房真正让人不安的，也果然不只是缺页，而是缺页前后的那些名字。你顺着旧账往里翻时，发现有几个人被反复记过，又被反复划掉，像是同一批人被换过用途、改过归处，最后连去向都被一并抹平。

这已经不是谁手脚不干净那么简单了。账册上留下的痕，不像在遮灵石，也不像在遮偷药，更像在遮“哪些人本该还活着，后来却被算作别的用途”。

=> [先记下被反复划掉的人名，把这份账缝攥在手里]
你没有急着把缺页拼完整，而是先把那些被反复划掉的人名和时辰牢牢记住。只要这些地方对不上，药房这条线就不是传闻，而是一份真能继续往里翻的账。$获得状态: 已进入旧账室, 增加标记: 发现账册被动过手脚, 线索碎片: +1, 天魔经验: +1, 获得卡牌: 缺页人名$

=> [把族兄留下的时辰和这些缺口对一遍，先确认他是不是也被记进过账里]
你先拿族兄留下的时辰去对那几处缺口，结果越看越心惊。那几页被撕走的地方，不只是碰巧空了一段，而像是正卡在一批人被转走、被替换、或被改作别用的节点上。族兄的死，恐怕从一开始就和这本账分不开。$获得状态: 已确认族兄之死有异, 增加标记: 发现账册被动过手脚, 线索碎片: +1, 天魔经验: +1, 获得卡牌: 缺页人名$
